# JSON Shema
A Java library to generate JSON Schema files.

## Installation

Add the following code to your **pom.xml**:

```xml
    <repositories>
        <repository>
            <id>jitpack.io</id>
            <url>https://jitpack.io</url>
        </repository>
    </repositories>
```

and the dependency:

```xml
    <dependency>
        <groupId>com.github.mauriciotogneri</groupId>
        <artifactId>json-schema</artifactId>
        <version>1.0.0</version>
    </dependency>
```
